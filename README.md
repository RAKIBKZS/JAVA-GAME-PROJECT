# Tic-Tac-Toe (Java Swing)

A Java-based Tic-Tac-Toe game with a graphical user interface built using Swing. The game supports both Player vs Player and Player vs AI modes with multiple difficulty levels.

## Features

- 3×3 interactive game board
- Player vs Player mode
- Player vs AI mode
- Three AI difficulty levels:
  - Normal (Random moves)
  - Standard (Blocking strategy)
  - Hard (Minimax algorithm)
- Score tracking system:
  - Player wins
  - Robot wins
  - Draws
- Restart functionality
- Keyboard shortcut to restart after game over
- Visual highlighting for winning and tie states

## Technologies Used

- Java
- Java Swing
- Java AWT
- Object-Oriented Programming (OOP)
- Minimax Algorithm (AI)


## How to Run
Crete these files in vscode.Then run  the code from App.java file.Then you will be able to play the game.

## How to Play

1. Launch the application.
2. Select game mode:
   - Player vs Player
   - Player vs Robot
3. If Robot mode is selected, choose AI difficulty.
4. Players take turns placing X and O.
5. The first player to align three marks horizontally, vertically, or diagonally wins.
6. If the board fills without a winner, the game ends in a draw.

---

## AI Levels

- **Normal:** Chooses random available moves.
- **Standard:** Attempts to block opponent winning moves when possible.
- **Hard:** Uses the Minimax algorithm for optimal gameplay.


## Controls

- Click a cell to make a move
- Press any key after game over to restart
- Use the Restart button to reset the game anytime
